# Paperclips

[Demo](https://adrienloup.github.io/paperclips/)

## Requirements

- `node` : `^18.18.2`
- `npm` : `^10.9.1`

## Installation

```bash
nvm use
npm install
```

## Development

```bash
npm run dev
```

## Production

```bash
npm run build
npm run preview
```

## Checking Code Quality

```bash
npm run lint
npm run lint:fix
```
